Kein unsichtbarer Service

MeatPad lädt keinen Code hoch, um Editor-Informationen bereitzustellen.

Es erkennt bereits auf Ihrem Mac installierte Sprachserver und startet sie im Projekt. Wenn kein Server vorhanden ist, funktionieren die normale Bearbeitung und die lokale Vervollständigung weiterhin.

Die Mac App Store-Sandbox kann keine vom Benutzer installierten Sprachserver ausführen. Die vollständige LSP-Unterstützung gehört daher zum notariell beglaubigten Direkt-Download-Build.
