MeatPad은 빠른 메모, Markdown 및 실제 코드 프로젝트를 위한 로컬 우선 홈입니다. 작업 내용을 브라우저로 옮기지 않고도 모든 것을 검색하고, 실제 구문을 접고, 모든 색상을 조정하고, 자신만의 도구를 가져올 수 있습니다.

본격적인 도구로 자라는 노트

간단할 때는 일반 텍스트. 필요할 때는 프로젝트.

폴더를 열면 MeatPad은(는) 파일 트리, 탭, 구문 인식 구조, 프로젝트 검색 및 눈앞에 있는 문서를 이해하는 도구를 갖춘 전문 편집기가 됩니다.

01 / 프로젝트 작업공간

스레드를 잃지 않고 코드베이스를 통해 이동합니다.

폴더를 탐색하고, 여러 파일을 열어두고, 방해받지 않는 기본 창에서 편집하세요.

02 / 코드처럼 읽는 Markdown

하나의 실제 편집기에서 산문과 울타리 코드를 작성하세요.

Markdown 구조 및 내장 언어는 줄 번호 및 기본 텍스트 동작과 함께 읽기 쉬운 상태로 유지됩니다.

보이지 않는 서비스 없음

MeatPad은 편집자 정보를 제공하기 위한 코드를 업로드하지 않습니다.

Mac에 이미 설치된 언어 서버를 감지하고 프로젝트 내에서 실행합니다. 서버가 없으면 일반 편집 및 로컬 완성이 계속 작동합니다.

Mac App Store 샌드박스는 사용자가 설치한 언어 서버를 실행할 수 없습니다. 따라서 전체 LSP 지원은 공증된 직접 다운로드 빌드에 속합니다.

로컬은 운영상 로컬을 의미합니다.
