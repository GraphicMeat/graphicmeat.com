Nessun servizio invisibile

MeatPad non carica codice per fornire intelligenza all'editor.

Rileva i server linguistici già installati sul tuo Mac e li avvia all'interno del progetto. Se un server è assente, la modifica ordinaria e il completamento locale continuano a funzionare.

La sandbox del Mac App Store non può eseguire i server delle lingue installati dall'utente. Il supporto completo LSP appartiene quindi alla build autenticata con download diretto.

Rendi tuo l'editor

Ogni token ha un colore. Lo scegli tu.

Inizia con un tema integrato bilanciato, quindi regola lo sfondo dell'editor, il testo, i commenti, le parole chiave, le stringhe, i numeri, i tipi e altro ancora. L'anteprima si aggiorna mentre lavori.

Locale significa operativamente locale

Nessun account. Nessun motore di sincronizzazione. I dati restano tuoi.

La schermata sulla privacy di MeatPad dice cosa fa l'app in un linguaggio semplice. Le note e le impostazioni rimangono su questo Mac. L'esportazione crea file ordinari. Elimina tutto rimuove i dati locali solo dopo una conferma esplicita.
